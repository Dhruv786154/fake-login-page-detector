# Fake Login Page Detector

This project is a frontend-based cybersecurity demonstration tool that helps identify potentially risky or fake login pages using simple URL analysis techniques.

## Project Overview
The application analyzes a user-provided website URL and calculates a basic risk score based on:
- HTTPS availability
- Presence of suspicious keywords commonly used in phishing attacks

If a website is considered risky, a demo login interface is shown along with a security warning.

## Features
- URL risk analysis
- HTTPS verification
- Suspicious keyword detection
- Risk score display
- Educational fake login warning

## Tech Stack
- HTML
- CSS
- JavaScript

## Disclaimer
This project is created strictly for educational and awareness purposes.
It does not perform real phishing detection and should not be used as a production security tool.
Never enter real credentials into demo applications.
